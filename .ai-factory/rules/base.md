# Base rules

- Писать артефакты на русском, технические имена классов/эндпоинтов оставлять на английском.
- Не добавлять лишние фреймворки, если задача решается стандартными средствами Laravel.
- Внешние контракты фиксировать в docs и Laravel contract/value object classes.
- Ошибки API возвращать в едином формате: `error`, `message`, `request_id`.
- Любое изменение auth-flow должно сопровождаться тестом или smoke-сценарием.
