- `verified` / `unverified`
- `active` / `revoked` / `expired`
- `api-only` / `web-login` / `both`
- `success` / `warning` / `danger` / `neutral`

## Empty states

Every list page should include a contextual empty state:

- No social accounts linked yet.
- No user tokens created yet.
- No sites connected yet.
- No audit events match filters.
